 # 루트박스와 키(Lootboxes and Keys)

루트박스는 **시즌 IV부터 라이브**입니다! 키를 획득해 루트박스를 열고, **Emblems, 스킨, 기타 희귀 아이템**을 손에 넣으세요.

### 루트박스에는 무엇이 들어 있나요?

각 루트박스는 **정확히 1개 아이템**을 제공합니다. 가능한 아이템은 아래와 같습니다.

- [Emblems](emblems) **5단계 희귀도**(낮은→높은): **Common, Uncommon, Rare, Legendary, Limited Edition**, 그리고 **공개 전(미공개) 3개 클래스**. **Limited Edition** 및 **미공개 Emblems**는 **거래 가능**, 나머지는 **계정 귀속**입니다.
  - 각 4개 클래스별 **최대 50개**의 **Limited Edition** Emblem이 **선착순(FCFS)** 으로 루트박스에서 열립니다.
  - 각 3개 미공개 클래스별 **최대 100개**의 Emblem도 **FCFS**로 열립니다.
- **Pudgy 스킨 30개**(선착순)
- **2-Chip 또는 5-Chip 랭크 바우처**
- 소수의 **Genesis Ace**들

### 루트박스는 어떻게 여나요?

루트박스를 열려면 **키(Keys)** 가 필요합니다. **Gold Key**와 **Silver Key** 두 종류가 있습니다.

- **Gold Key**
  - **직접 구매** 또는 **시즌 패스 보상**으로 획득
    - **비용: 999 Sparks**
  - **Gold Key 1개 = 루트박스 1개 오픈**
  - **Platinum Pass 구매 시 Gold Key 1개 무료 제공**
- **Silver Key**
  - **게임 플레이**로 획득
    - **랭크**
      - **게임 내 최고 수익자(Top earner)** → Silver Key 1개
      - **스트레이트 플러시 승리자** → Silver Key 2개
    - **캐주얼**
      - **게임 내 최고 수익자**에게 **낮은 확률**로 Silver Key 1개
  - Silver Key로 루트박스를 열 때 **주간 소모량**:
    - **1번째**: Silver Key **2개**
    - **2번째**: **3개**
    - **3번째**: **4개**
    - **4번째 이후**: **5개**
    - **매주 월요일에 비용 초기화**
