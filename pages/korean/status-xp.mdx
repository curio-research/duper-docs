# 상태와 XP

듀퍼의 진행도 시스템은 명예 아바타 프레임을 부여하고, 시즌 패스 보유자에게 **상태(Status)**에 따른 독점 혜택을 해금합니다.

![XP and Status](public/XP-status.png "XP and Status")

참고: 상태는 **게임플레이와 칩 잔고**의 함수이며, 전형적인 랭킹 시스템처럼 **성과(승패)**에 묶이지 않습니다. 따라서 누구나 **Super Duper** 상태에 도달할 수 있습니다.

## XP 계산

XP는 진행도의 기본 단위입니다. 진행도 바에는 최대 **8,000 XP**까지만 표시되며, 초과분은 누적되지만 바에는 보이지 않습니다.

XP를 얻는 방법은 두 가지입니다:

### 볼륨(Volume)에서 얻는 XP

플레이한 모든 게임의 **입장비 합계** + **Gold Key 구매액**  
- Ranked 입장비: **10 Chip = 10 XP**
  - 예: 레이즈 없이 Ranked 1판 → **10 XP**
- Casual 입장비: **2 Chip = 2 XP**
- Gold Key 구매: **USD 기준 구매 금액**

지난 시즌의 Volume XP에는 **0.7배** 할인 계수가 적용됩니다. Volume XP에는 **획득 제한이 없습니다.**

### 칩 잔고에서 얻는 XP

칩을 예치하면 **동일 수량의 XP(최대 2,500)** 를 획득합니다.  
예: 잔고 1,500 Chips → **1,500 XP**. 이 XP는 칩 잔고에 따라 **실시간 업데이트**됩니다.

## 상태 티어

각 상태(Status)는 시즌 전반에 걸쳐 갱신되는 요구 XP를 가지며, **어느 티어의 시즌 패스라도 보유 중**이면 해당 혜택을 해금합니다. 상위 티어는 하위 혜택을 유지합니다.

- **5000+ XP → Super Duper**
  - 해금: Ranked **시작 골드 +1**
- **3000+ XP → Gold**
  - 해금: Ranked **시작 에너지 +1**
- **1200+ XP → Silver**
  - 해금: Ranked에서 **순이익이지만 최고 수익자는 아닐 때 50% 확률**로 Silver Key
- **500+ XP → Bronze**
  - 해금: Ranked에서 **순이익이지만 최고 수익자는 아닐 때 25% 확률**로 Silver Key
- **0+ XP → Iron** (인게임 아바타 프레임 없음)

또한 **8,000 XP**에 도달하면 일회성 **Duper Maxi soulbound** 배지를 획득합니다.

### 상대 난이도 점수(RDS)

**RDS(Relative Difficulty Score)** 는 한 게임에 참가한 모든 플레이어의 평균 랭크 값을 계산해 산출됩니다. 각 랭크에는 수치가 있습니다:

- 브론즈 = 1, 실버 = 2, 골드 = 3, 슈퍼 듀퍼 = 4

예를 들어, 브론즈 3명, 골드 1명, 슈퍼 듀퍼 1명이 있는 게임의 RDS는 (1+1+1+3+4) ÷ 5 = 2입니다. 이 점수에 따라 **하위 랭크 플레이어는 보너스**, **상위 랭크 플레이어는 페널티** XP를 받습니다.

그 게임에서 브론즈가 단독 최대 수익자라면, 2 - 1 = 1 XP 보너스를 받습니다. 골드는 3 - 2 = 1 XP 페널티, 슈퍼 듀퍼는 4 - 2 = 2 XP 페널티를 받습니다. 이는 서로 다른 랭크 간의 공정한 경쟁을 보장합니다.
