# Introduction

Duper is a competitive onchain game and esports platform. The core game is a wagering social war game. The game uses [Keystone](https://www.keystonesdk.com) technology, and the game contract is currently deployed on Arbitrum One.

This is the official documentation site for Duper. Currently, it contains a Quick Start guide and a comprehensive Game Wiki. It also covers game modes, XP, and rewards.

Duper is currently in alpha. If you're here, you are early.
