# Duper

Duper is an onchain wagering social war game. It is a competitive game where you can win real money.

The game uses [Keystone](https://www.keystonesdk.com) technology, and the game contract is currently deployed on Arbitrum One. The documentation contains a Quick Start guide and a comprehensive Game Wiki. It also covers the metagame including rewards, seasons, and beyond.

Duper is currently in alpha. If you're here, you are early.
