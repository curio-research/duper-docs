# Движение

Перемещение войск по своим собственным клеткам важно как для атаки, так и для обороны.

Чтобы переместить войска, сначала выберите исходную клетку. Если вы хотите переместить только часть войск с клетки, измените выбор в панели движения, которая появляется справа от карты. Затем перетащите клетку на целевую, которая будет подсвечена синим, если перемещение допустимо. Все юниты, способные к движению, имеют одинаковую скорость.

Правила движения:

- Игрок может перемещать только свои юниты по своим клеткам.
- Игроки могут делить войска при перемещении (например, переместить 1 пехотинца и 3 самолёта из 5 пехотинцев и 5 самолётов).
- Юниты перемещаются по одной клетке за раз, то есть нельзя перемещаться через чужую клетку на свою. Исключение — самолёты.
