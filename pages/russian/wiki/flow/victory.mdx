# Условия победы

В Duper есть 4 способа выиграть:

1. Tile Count Victory (Победа по количеству клеток): Игрок владеет необходимым количеством клеток для победы. Конкретное число зависит от карты и отображается в панели целей.
2. Straight Flush Victory (Победа с флеш-стритом): Игрок держит 5-card straight flush.
3. Last Man Standing Victory (Победа последнего выжившего): Игрок — единственный, кто не был уничтожен и не покинул игру.
4. Round Limit Victory (Победа по окончании раунда): После 12-го раунда побеждает игрок с наибольшим количеством клеток. В случае ничьей выигрывает игрок, который ходит раньше в порядке ходов.

Как только один из условий выполнен, сессия заканчивается, и распределяются выплаты.
