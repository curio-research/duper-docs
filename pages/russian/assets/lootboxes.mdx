# Лутбоксы и Ключи

Лутбоксы доступны с Сезона IV! Получайте эмблемы, скины и другие редкие предметы, приобретая ключи для открытия лутбоксов.

### Что внутри лутбокса?

Каждый лутбокс открывает ровно 1 предмет. Ниже перечислены все возможные предметы из лутбокса:

- 5 уровней [Эмблемы](emblems) по возрастанию редкости: Common, Uncommon, Rare, Legendary и Limited Edition, а также 3 пока нераскрытых класса эмблем. Эмблемы Limited Edition и нераскрытые эмблемы доступны для торговли; остальные привязаны к аккаунту.
  - Up to 50 Limited Edition Emblems for each of the 4 classes can be opened from Lootboxes, on a first-come, first-served basis
  - До 50 эмблем Limited Edition каждого из 4 классов можно открыть из лутбоксов по принципу FCFS
- 30 скинов Pudgy по принципу FCFS
- Ваучер Ranked на 2 или 5 Chips
- Небольшое количество Genesis Ace

### Как открыть лутбокс?

Для открытия лутбоксов требуются ключи. Существуют два типа ключей: Gold Keys и Silver Keys.

- Gold Key
  - Получается через прямую покупку или бонусы Сезонного пропуска
    - Стоимость: 999 Sparks
  - 1 Gold Key открывает 1 лутбокс
  - Покупка Platinum Pass даёт один бесплатный Gold Key
- Silver Key
  - Получается через игровой процесс
    - Ranked
      - Топ-игрок каждой игры получает 1 Silver Key
      - Победитель со Стрит-Флешем получает 2 Silver Keys
    - Casual
      - Топ-игрок каждой игры имеет небольшой шанс получить 1 Silver Key
  - Открытие лутбоксов с использованием Silver Keys требует разного количества ключей каждую неделю:
    - 1-й лутбокс: 2 Silver Keys
    - 2-й: 3 Silver Keys
    - 3-й: 4 Silver Keys
    - 4-й и последующие: 5 Silver Keys
    - Стоимость сбрасывается каждый понедельник
